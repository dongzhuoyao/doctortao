# 序言

谁的人生不迷茫，大家都是半斤八两，如果你硬着头皮挺上去，你会发现---



{% hint style="info" %}
别人也只是在硬着头皮挺着而已。
{% endhint %}

学会本心法不会让你发表多牛逼的paper，

只是让你在学术界这个脑力竞争非常激烈的地方能够保持一颗平和的心态，

对于自己博士期间，甚至后半段的成事都会有巨大的作用。
